Image Functions

Redondear Imágen

Crea una imagen con bordes redondos

Uso:

const img = await loadImage('image.png'); const borderRadius = 20; const strokeColor = "#ff0000"; const strokeWidth = 4 const quality=1; roundImage(img,borderRadius, quality, strokeColor, strokeWidth);

Imágen circular

Crea una imagen con circular

Uso:

const img = await loadImage('image.png'); const strokeColor = "#ff0000"; const strokeWidth = 4 const quality=1; processCircularImage(img, quality, strokeColor, strokeWidth);
